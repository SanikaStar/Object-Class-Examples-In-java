**Java Object Class Programs**

This repository contains Java programs demonstrating Object-Oriented Programming (OOP) concepts, especially focusing on the Object class, equals(), toString(), and object comparison.

The repository includes the Student2 class and its driver file StudentDriver2.java which showcases how objects behave when printed, compared, and checked for equality.
